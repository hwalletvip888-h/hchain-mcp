# /audit — 安全审计

对指定代币或交易策略执行全方位的安全审计。

## 参数
- `$1`: 审计目标 (代币地址 或 "portfolio" 审计持仓)
- `$2`: 链ID (可选)

## 审计流水线

### 代币安全审计
并行启动:
- `risk-assessor` → 完整4维风险评分 + 所有安全检测
- `social-analyst` → 负面新闻、社区预警
- `signal-scout` → 该代币的聪明钱动向

Orchestrator 汇总 → 输出审计报告:
- 🛡️ 合约安全评级
- 💧 流动性分析
- 👥 持仓分布图
- 📰 社媒风险评估
- 💰 聪明钱态度
- ✅ 最终裁决: 安全 / 谨慎 / 危险

### 持仓审计
- `portfolio-tracker` → 获取全部持仓
- `risk-assessor` → 逐代币快速风险扫描
- 输出: 持仓风险矩阵 + 建议调整

## 输出格式
完整 Markdown 审计报告，含风险矩阵表格和操作建议。

## 写作原则
这是安全审计，宁可误报（假阳性）不可漏报（假阴性）。
