# /research — 代币深度调研

对指定代币执行全方位的深度调研。

## 参数
- `$1`: 代币地址
- `$2`: 链ID (可选，默认 '1')

## 流水线

### 阶段 1: 并行数据收集
同时启动 4 个 Agent:
- `market-analyst` → 价格、K线、基本信息、流动性
- `risk-assessor` → 4维风险评分、持仓分布、Bundle检测
- `social-analyst` → 社媒情绪、KOL、新闻
- `signal-scout` → 聪明钱是否在买、信号强度

### 阶段 2: 综合评级
Orchestrator 汇总 4 份报告，输出综合调研报告:
- 代币基本信息
- 行情分析 (价格/K线/流动性)
- 风险评分 (0-100 + 明细)
- 社媒情绪 (分数 + 趋势)
- 聪明钱信号 (有/无)
- **综合评级**: A(优秀) / B(良好) / C(一般) / D(谨慎) / F(危险)
- **操作建议**: 买入 / 观望 / 回避

## 写作原则
请并行启动 4 个 Agent，汇总结果后输出 Markdown 格式的调研报告。
